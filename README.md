# Holistic-AI-framework-for-personalized-walktime-recommendation

## Project Overview

This project presents a Holistic AI Framework designed to predict stress level and provide personalized walktime recommendations using machine learning techniques. The system aims to support healthier lifestyle decisions through intelligent health analytics.

## Features

* Stress Level Prediction
* Personalized Walk-Time Recommendation
* Health Monitoring Support
* Weather-Based Assistance
* Interactive Chatbot Support
* Heart rate measurement
* Tracker 

## Technologies Used

* Python
* Streamlit
* Machine Learning
* HTML
* CSS
* Scikit-learn
* Pandas
* NumPy

## Machine Learning Models

### Stress Level Prediction

* Random Forest Classifier

### Walk-Time Recommendation

* HistGradientBoosting Regressor

## Project Structure

* app.py – Main application
* train_stress.py – Stress model training
* train_walk.py – Walk-time model training
* chatbot.py – Chatbot functionality
* weather.py – Weather integration
* heart_rate.py – Heart rate processing
* tracker.py – Health tracking

## Future Enhancements

* Real-time wearable device integration
* Advanced health analytics dashboard
* Mobile application development
* Advanced AI based chatbot with NLP
