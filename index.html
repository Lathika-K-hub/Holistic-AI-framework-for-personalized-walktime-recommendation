About US

This is a Personalized Walking Time Recommendation system Using a Conversational Chatbot” focuses on improving health and fitness through artificial intelligence. The system analyzes personal health data such as age, BMI, heart rate, sleep patterns, stress levels, and real-time weather conditions to recommend the most suitable walking time and duration. Unlike traditional fitness apps that give general suggestions, this project provides personalized recommendations using machine learning techniques. A conversational chatbot interacts with users and explains walking suggestions. The main goal of this project is to encourage safe, healthy, and sustainable walking habits while improving overall well-being.
